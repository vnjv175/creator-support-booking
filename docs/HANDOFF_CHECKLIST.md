# Referral Program Handoff Checklist

Use this checklist when onboarding a new team member to manage the referral program.

---

## Access Setup

### Required Accounts
- [ ] Google account with access to [Referral Data Sheet](https://docs.google.com/spreadsheets/d/1kCFAavAsfbb3p-cNTxudFh6eMd3uOHxkYJrAZqgCpRE)
- [ ] Access to intake@arunatalent.com email (or forwarding set up)
- [ ] Access to payment platform for revenue data

### Verify Access
- [ ] Can view and edit the Google Sheet
- [ ] Can receive emails at intake@arunatalent.com
- [ ] Can access payment platform reports

## Hands-On Training

### Practice Tasks (with supervision)

#### Task 1: Review a Referrer Application
- [ ] Find a recent application in the Applications tab
- [ ] Walk through the decision criteria in SOP 01
- [ ] Explain what action would be taken

#### Task 2: Generate a Referrer Code
- [ ] Create a test code following SOP 06
- [ ] Verify it doesn't exist
- [ ] Add to Referrers sheet (then delete - this is practice)

#### Task 3: Add a Model
- [ ] Walk through adding a model to the Models sheet
- [ ] Explain the column format
- [ ] Show how monthlyRevenue is formatted

#### Task 4: Update Monthly Revenue
- [ ] Show where to get revenue data
- [ ] Practice appending to monthlyRevenue column
- [ ] Verify format is correct

#### Task 5: Send an Email
- [ ] Find a template in email-templates.md
- [ ] Customize it with real data
- [ ] Show how to send (don't actually send during training)

---

## Shadowing Period

### Week 1: Shadow Mode
- [ ] New team member observes all tasks
- [ ] They take notes on questions
- [ ] Daily check-ins to answer questions

### Week 2: Supervised Execution
- [ ] New team member handles tasks with review before sending/saving
- [ ] Trainer reviews all work before completion
- [ ] Document any unclear processes

### Week 3: Independent with Review
- [ ] New team member works independently
- [ ] End-of-day review of all actions taken
- [ ] Escalation path is clear

### Week 4+: Fully Independent
- [ ] Weekly check-ins
- [ ] Available for questions as needed
- [ ] Monthly process review
