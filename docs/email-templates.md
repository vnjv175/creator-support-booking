## Tips for Using Templates

1. **Personalize**: Always replace [BRACKETS] with actual values
2. **Proofread**: Double-check names and codes before sending
3. **Timing**: Send welcome emails within 24 hours of approval
4. **Tone**: Keep it friendly but professional
5. **Follow-up**: If no response in 3-5 days, send one follow-up
